#### VBX脚本的使用

* 运行脚本需要系统已经安装了 [NScript](https://gitee.com/milaoshu1020/NScript)([下载地址](https://gitee.com/milaoshu1020/NScript/releases));
* 安装完毕以后,双击VBX脚本即可运行;
* 打开任务管理器,结束"wnscript.exe"进程可以终止脚本运行;
* 你可以用记事本或者其他编辑器查看甚至编辑VBX文件;
